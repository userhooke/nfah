# NFAH VS Code Support

Minimal syntax highlighting for `.nfah` files.

## Run Locally

Open this folder in VS Code:

```bash
code editor-support/vscode
```

Press `F5` to launch an Extension Development Host, then open a `.nfah` file.

## Install Locally

Copy or symlink this folder into your VS Code extensions directory:

```bash
ln -s "$PWD/editor-support/vscode" "$HOME/.vscode/extensions/nfah-vscode"
```

Restart VS Code.
